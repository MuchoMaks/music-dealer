Connexion via le flux officiel PKCE — aucun mot de passe ne transite par cette page, tout passe par Spotify. Lit tes propres playlists et tes top tracks, puis prépare la liste pour le moteur de découverte.

À héberger, pas à ouvrir dans le chat. Le login Spotify a besoin d'une vraie URL enregistrée comme redirect URI : ça ne peut pas aboutir dans l'aperçu Claude. Déploie ce fichier (Vercel / Netlify / GitHub Pages) ou ouvre-le via http://127.0.0.1:PORT avec un petit serveur local. localhost n'est plus accepté par Spotify — utilise 127.0.0.1 ou du HTTPS.
Configuration (une fois)
  1. Va sur developer.spotify.com/dashboard → Create app.
  2. Dans Redirect URIs, colle l'URL ci-dessous (celle de cette page), puis enregistre.
  3. Copie le Client ID de l'app et colle-le ici.
